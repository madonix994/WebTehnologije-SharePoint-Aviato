Uvod u SharePoint


MS SharePoint je aplikacija firme Microsoft koja ima veoma široku primenu u oblasti IT. Koristi se za kreiranje Web lokacija. Pored toga, Microsoft SharePoint se koristi kao bezbedno mesto za skladištenje, organizovanje, deljenje, i pristupanje informacijama na bilo kom mestu i sa gotovo svakog uređaja. Sve što je potrebno je web browser (Google Chrome, Internet Explorer, Mozilla, Edge,...) i naravno pristup internetu.



Postoji više različitih izdanja ove aplikacije. Neka od njih su:

Microsoft SharePoint Server  Microsoft SharePoint Standard  Microsoft SharePoint Enterprise  SharePoint Online




MS SharePoint Server


Pošto je Microsoft SharePoint generalno veoma širok pojam, kao predmet razmatranja u ovom seminarskom radu uzećemo MS SharePoint Server i demonstrirati na primeru primenu i funkcionalnost MS SharePoint-a. Postoje MS SharePoint Server Online u okviru Office 365 Paketa na Cloud-u, i lokalni (On-premises) MS SharePoint Server. Takođe je moguće sastaviti ove dve vrste u Hibridni MS SharePoint Server.




Lokalni (on-premises) Microsoft SharePoint Server služi da modernizuje lokalnu računarsku infrastrukturu, poboljšava performanse, daje dodatnu pouzdanost pri radu, dok istovremenu pruža osnovu za buduće inovacije. MS SharePoint Server radi na Cloud-u, tako da možemo reći da je veoma fleksibilan što se tiče toga. Zaštita podataka je veoma dobra. MS SharePoint omogućava čitav niz karakteristika i sposobnosti koje pomažu u zaštiti osetljivih podataka i da omoguće pravim ljudima da u pravo vreme na pravom mestu imaju pristup podacima. U našim tutorijalima moćićete da naučite nešto više konkretno o MS SharePoint Server-u 2010 i 2016.

MS SharePoint Server 2016

Microsoft SharePoint Server Online je pretežno sadržan na Cloud-u u Office 365 paketu u kome postoji mogućnost dobijanja pristupa sistemu Exchange, uslugama OneDrive for Business i Skype for Business, kao i Office klijentima i Web aplikacijama, ali može se koristiti i kupiti u oblaku kao stamostalna usluga. Služi da se lako naprave sajtovi u timu, portali za otkrivanje i deljenje sadržaja, kao i saradnja na njemu sa kolegama u organizaciji i izvan nje sa bilo kog uređaja.SharePoint Online pruža moćne funkcije sistema SharePoint bez povezane dodatne podrške za samostalno upravljanje infrastrukturom.
